# Notebooks

Use notebooks only for exploratory analysis and short-lived experiments.

- Promote reusable logic into `jobs/`, `libs/common/`, or `sql/`.
- Do not treat notebooks as production pipelines.

В ноутбуках уже была проведена работа с датасетами:
- удалены дубликаты
- очищены от нулевых значений
- изменен формат некоторых данных

С готовым датасетом был проведен некоторый анализ данных:
- оценка распределения по некоторым показателям (`пол`, `лояльность`, `статус`)
- построен ABC-анализ по `sale_price`
- анализ прибыльности по группам ABC и доля оттока
- среднее количество заказов клиента
- средний чек по категориям
- средняя продолжительность жизни клиента
- построен XYZ-анализ по сумме покупок, количеству заказов и числу клиентов
